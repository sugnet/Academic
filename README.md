# createQuarto_website
This is the template to create your own website.
The material will be used in Part 4 of the MDAG 2025 workshop on 25 November 2025 at the South African Statistical Association (SASA). <br>
[MDAG 2025 workshop website](https://muvisu.github.io/SASA2025_MDAG/)

The following repositories were used as inspiration to create the content for this workshop:
-   [Di Cook](dicook.github.io/dicook-quarto/)
-   [JSM 2024 repo](https://github.com/mine-cetinkaya-rundel/quarto-world-of-possibilities-jsm24)
-   [Joyce Robbins repo](https://github.com/jtr13/website-template)
